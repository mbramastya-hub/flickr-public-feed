# Codebase-Express
A back end project template using Express.js as a Node.js framework.

## Version

Current app version is on v1.0.

## Usage

```
# Install dependencies
npm install

# Run in development
npm run dev

# Run in production
npm start

# Initialize mongodb
mongod

```
